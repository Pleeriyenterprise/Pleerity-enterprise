# How to upload documents

In Compliance Vault Pro you can upload documents from:

1. **Documents page** – Use "Upload" to add a file. You can link it to a property and, where applicable, to a requirement type (e.g. Gas Safety, EICR).
2. **Property view** – From a property’s requirements, use the upload action for that requirement to attach evidence.
3. **Vault** – Client vault uploads are stored and can be matched to requirements by an admin or by the system where applicable.

Supported formats typically include PDF and common image types. Keep file names clear so you can identify the certificate later.

After upload, the system may suggest extracted details (e.g. expiry date) for you to confirm. Always verify the details before confirming.
