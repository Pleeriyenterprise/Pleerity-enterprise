# Assistant Knowledge Base

This folder contains curated markdown used by the Compliance Vault Pro Assistant to answer questions about the portal, certificates, uploads, and compliance concepts.

- **Information only** – Content is for guidance only and does not constitute legal advice.
- **No compliance verdict** – The assistant describes what the portal shows and suggests actions; it does not determine legal compliance.
- **If uncertain** – Users should upload or confirm evidence and seek professional advice where needed.

Files: certificates_overview.md, how_to_upload.md, how_scoring_works.md, glossary.md, support_and_escalation.md, data_sources_and_limits.md.
